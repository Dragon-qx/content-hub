## 1. 产品概述

ContentHub 是一个面向内容运营团队的一站式多平台管理与发布平台。你可以：

- **统一管理** 在微信、抖音、小红书、B站、微博、Twitter、YouTube 等 8 大平台的账号
- **集中创作** 使用 Markdown 编辑器撰写内容，AI 辅助优化标题、提取标签、审核质量
- **一键发布** 内容自动适配各平台规则（字数、图片、视频限制），支持定时和批量发布
- **数据驱动** 实时追踪核心指标，自动检测异常波动并告警
- **互动聚合** 统一收件箱管理所有平台的评论和私信
- **团队协作** 支持多级审批流、角色权限管理和操作审计

## 2. 快速开始

### 2.1 注册与登录

1. 打开 ContentHub，点击「注册」
2. 输入邮箱、密码（≥8 位，含大小写字母和数字）
3. 注册成功后自动登录，进入仪表盘

### 2.2 设置双因素认证（TOTP）

为了账户安全，建议开启两步验证：

1. 进入 **设置 → 安全 → 两步验证**
2. 点击「开启」，用 Google Authenticator 或类似 App 扫描二维码
3. 输入 App 显示的 6 位验证码完成绑定
4. 之后登录时需要同时输入密码和 TOTP 码

### 2.3 绑定平台账号

1. 进入 **账号管理** 页面
2. 点击「绑定账号」，选择平台（如微信公众号）
3. 选择授权方式：
   - **OAuth 授权**（推荐）：跳转到平台官方页面授权，安全便捷
   - **手动输入**：直接填入 API Key / Secret（如平台不支持 OAuth）
4. 授权成功后，账号出现在列表中，状态为「已连接」

## 3. 仪表盘

登录后默认进入仪表盘，展示：

- **核心指标卡片**：总粉丝数、总互动量、发布内容数、活跃账号数
- **趋势图**：近 30 天粉丝增长与互动量变化
- **待办事项**：待审批内容、即将发布的内容、异常告警
- **快捷操作**：新建内容、绑定账号、查看报告

## 4. 账号管理

### 4.1 查看账号

- 列表展示所有已绑定平台账号，含平台图标、昵称、粉丝数、健康度
- 支持按平台、健康度、状态筛选

### 4.2 账号健康度

系统根据以下维度评估账号健康度（0-100 分）：

- 最近发布频率
- 粉丝增长趋势
- 互动率
- 账号年龄

健康度低于 60 分会触发告警。

### 4.3 同步数据

点击账号上的「同步」按钮，手动拉取平台的最新指标数据。

### 4.4 解绑账号

在账号详情页点击「解绑」，确认后移除绑定（不会删除已发布内容的历史数据）。

## 5. 内容创作

### 5.1 创建内容

1. 点击「新建内容」进入编辑器
2. 输入标题和正文（支持 Markdown 语法）
3. 添加标签（可手动输入或让 AI 自动提取）
4. 选择内容类型：文章 / 图文 / 视频 / 动态
5. 上传或从媒体库选择图片/视频

### 5.2 Markdown 编辑器

- **双栏模式**：左侧编辑、右侧实时预览
- **格式工具栏**：快速插入标题、加粗、链接、图片、列表等
- **图片上传**：拖拽或点击上传，自动插入 Markdown 语法
- **字数统计**：实时显示正文字数

### 5.3 AI 辅助写作

编辑器右侧面板提供 4 个 AI 工具：

| 工具 | 功能 |
|------|------|
| **标题优化** | 根据正文生成多个标题变体（中文/英文），提升点击率 |
| **标签提取** | 自动从正文提取关键词作为标签 |
| **内容审核** | 检查内容质量（长度、可读性、平台限制），给出评分和改进建议 |
| **文案变体** | 生成不同风格的文案：简短、详细、正式、社交化 |

### 5.4 内容适配预览

编辑内容时，系统自动按各平台规则实时预览适配效果：

- 超出字数限制的自动截断提示
- 图片/视频数量超出上限的裁剪提示
- 视频时长不足的警告

### 5.5 内容模板

- 从模板库选择预设模板快速创建内容
- 支持创建自定义模板（标题 + 正文 + 标签 + 内容类型）
- 模板按团队隔离，同团队成员可共享

### 5.6 版本管理

- 每次编辑自动保存为新版本
- 可查看历史版本、对比差异、回滚到任意版本

## 6. 审批工作流

### 6.1 提交流程

1. 内容编辑完成后，点击「提交审批」
2. 选择审批人（默认按团队角色路由）
3. 填写审批备注（可选）
4. 提交后内容状态变为「审批中」

### 6.2 审批操作

审批人在 **审批中心** 查看待办：

- **通过**：内容状态变为「已批准」，可进入发布环节
- **驳回**：内容退回「草稿」，附驳回原因，作者可修改后重新提交
- **批量审批**：支持多选批量通过/驳回

### 6.3 审批规则

- 审批人不能审批自己创建的内容
- 支持多级审批（配置于团队设置）
- 审批超时（默认 48 小时）自动提醒

## 7. 发布与调度

### 7.1 立即发布

已批准的内容可直接发布：

1. 选择目标平台账号
2. 预览适配效果
3. 点击「发布」，系统自动按平台规则适配后发布

### 7.2 定时发布

1. 选择发布日期和时间（支持到分钟）
2. 系统加入发布队列，到时自动执行
3. 支持多平台同时定时发布

### 7.3 批量发布

- 在内容列表多选，点击「批量发布」
- 为每个内容单独设置目标平台和发布时间
- 在 **调度中心** 统一管理所有定时任务

### 7.4 发布队列

调度中心展示：

- **待发布**：排队中的任务
- **发布中**：正在执行的任务
- **已完成**：发布成功的记录
- **失败**：发布失败的任务，支持手动重试

失败任务自动重试（指数退避策略：1s → 2s → 4s → ... → 60s 上限）。

## 8. 数据分析

### 8.1 数据总览

- 团队所有账号的聚合指标
- 支持按时间范围筛选（7天/30天/90天/自定义）

### 8.2 趋势分析

- 折线图展示指标随时间变化
- 支持多指标对比（如粉丝数 vs 互动量）

### 8.3 内容排行榜

- 按互动量、浏览量、粉丝增长等维度排名
- 自动标记 Top 10 和 Bottom 10 内容

### 8.4 异常检测

系统自动检测 5 类异常：

| 类型 | 说明 |
|------|------|
| 突降 | 指标较 7 日均值下降 >50% |
| 突增 | 指标较 7 日均值上升 >200% |
| 连续下滑 | 连续 3 个周期下降 |
| 断崖式下跌 | 单周期下降 >80% |
| 掉粉异常 | 单日掉粉 >5% |

检测到异常时自动向团队广播告警。

### 8.5 数据导出

- 趋势数据支持 CSV 导出
- 可导出指定时间范围的原始数据

## 9. 互动管理

### 9.1 评论收件箱

- 聚合所有平台账号下的评论
- 支持按平台、情感倾向、已回复/未回复筛选
- 情感分析自动标注：正面 / 中性 / 负面

### 9.2 回复评论

- 在收件箱中直接回复（B站、YouTube 支持原生回复）
- 支持快捷回复模板
- 回复后自动标记为「已回复」

### 9.3 舆情监控

- 设置关注关键词（团队级）
- 当评论包含关键词或强负面情感时，自动触发团队告警

### 9.4 私信管理

- 聚合各平台私信（B站原生支持）
- 按对话分组查看
- 支持回复私信（B站）

## 11. 团队管理

### 11.1 创建团队

- 点击「创建团队」，输入团队名称
- 创建者自动成为团队 Owner

### 11.2 成员管理

- 邀请成员：输入邮箱发送邀请
- 移除成员：在成员列表中移除
- 角色分配：

| 角色 | 权限 |
|------|------|
| Owner | 全部权限，管理团队成员 |
| Admin | 全部权限，不可管理成员 |
| Editor | 创建/编辑内容，不可审批自己 |
| Viewer | 只读访问 |

### 11.3 操作审计

- 所有关键操作记录审计日志
- 支持按操作类型、操作者、时间范围筛选
- 可导出审计报告

## 14. 常见问题

**Q: 发布失败怎么办？**
A: 进入调度中心查看失败原因。常见原因：平台 API 限流、凭证过期、内容不符合平台规范。修复后可手动重试。

**Q: 如何更换绑定账号？**
A: 先解绑旧账号，再绑定新账号。历史发布数据保留。

**Q: 审批人不在怎么办？**
A: 审批超时（48h）后系统自动提醒。Owner 可在设置中配置审批升级规则。

**Q: 支持哪些图片/视频格式？**
A: 图片：JPG、PNG、GIF、WebP；视频：MP4、MOV、AVI。单文件上限 100MB。

**Q: 如何接入新的平台？**
ContentHub 已内置 8 大平台适配器。如需接入其他平台，可在 Platform SDK 中实现适配器接口并注册到工厂。
